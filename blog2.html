2. Methods of Array.

Array is, in simple words, a collection of items. It is a data structure to store multiple elements in a single variables. The list is enclosed within square brackets seperated by commas. It has a very important usage in our everyday coding life.

There are fortunately many methods which makes our life easy dealing with the arrays. I will only be talking about four array methods here -

map()

Map method creates a new array with the result of the function called which is applied to every element in the array once.

filter()

This method creates a new array with the elements that passes or satisfies the condition passed within the function called.

sort()

This method is used to arrange elements of an Array in an ascending order and in a descending order.

concat()

This method is used to merge two differents arrays which forms a new Array.
